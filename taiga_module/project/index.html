Types

    • Closed User Story

      no implementation for this version
      Total number of closed user story information.
      Presentation:
      {
      "closed_user_stories": integer, Total number of closed user stories in a sprint.
      "name": string, The name of the sprint.
      "no_of_user_stories": integer, Total number of user stories in a sprint.
      "open_user_stories": integer Total number of open user stories in a sprint.
      }
      Formats:
      Sample:
      { "closed_user_stories": 42, "name": "my_name", "no_of_user_stories": 42, "open_user_stories": 42 }

    • Date

      no implementation for this version

      User
      ]
      }
      Formats:
